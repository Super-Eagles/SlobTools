# 文本与 Excel 工具集使用说明

> 所有脚本在同一目录下(D:\soft\mytools)，已配置好关联。
> 如要运行脚本 read_gbk.py 只需用命令 read_gbk

---

## 文件清单

| 文件 | 用途 |
|---|---|
| `encoding_utils.py` | 公共编码检测模块（其他脚本自动调用，无需直接使用） |
| `read_gbk.py` | 读取 / 搜索文本文件（自动识别 GBK / UTF-8） |
| `write_gbk.py` | 写入文本文件（自动识别并保持原有编码） |
| `gbktoutf8.py` | 批量转换目录：GBK → UTF-8 |
| `utf8togbk.py` | 批量转换目录：UTF-8 → GBK |
| `read_excel.py` | 读取 Excel 文件 |
| `write_excel.py` | 写入 Excel 文件 |

---

## read_gbk.py

自动检测文件编码（GBK / UTF-8 / ASCII / BOM），统一输出为 UTF-8。

### 读取

```bash
python read_gbk.py <文件>                        # 读取全文
python read_gbk.py <文件> --start 10 --end 20    # 指定行范围
python read_gbk.py <文件> --stats                # 文件统计（行数、大小、检测编码）
python read_gbk.py <文件> --out result.txt        # 结果保存到文件
python read_gbk.py <文件> --encoding gbk          # 手动指定编码（跳过自动检测）
```

### 搜索

```bash
python read_gbk.py <文件> --search "关键词"
python read_gbk.py <文件> --search "关键词" --context 3    # 显示前后 3 行
python read_gbk.py <文件> --search "\d{4}" --regex         # 正则搜索
python read_gbk.py <文件> --search "ABC" --case-sensitive  # 区分大小写
```

## write_gbk.py

自动检测文件编码并以相同编码写回，保证不破坏原有编码。
每次写入前自动备份（`.bak`），支持原子写入和写后校验。

### 内容来源（三选一）

```bash
--content "文本"       # 直接传入字符串
--content-file path    # 从 UTF-8 文件读取
# 不传任何参数          # 从 stdin 读取
```

### 六种操作模式

```bash
# append — 追加到末尾
python write_gbk.py <文件> --mode append --content "新增行"

# insert — 在指定行前插入（--start 为行号，1 = 最前面）
python write_gbk.py <文件> --mode insert --start 5 --content "插入内容"

# replace — 替换指定行范围
python write_gbk.py <文件> --mode replace --start 10 --end 15 --content "新内容"

# delete — 删除指定行范围
python write_gbk.py <文件> --mode delete --start 3 --end 7

# patch — 全局文本替换（支持正则，支持跨行匹配）
python write_gbk.py <文件> --mode patch --old "旧文本" --new "新文本"
python write_gbk.py <文件> --mode patch --old "\d+" --new "NUM" --regex
python write_gbk.py <文件> --mode patch --old "foo" --new "bar" --count 1  # 只替换第一处

# overwrite — 完全覆盖文件
python write_gbk.py <文件> --mode overwrite --content-file new.txt
```

### 常用选项

```bash
--encoding gbk/utf-8   # 手动指定编码（跳过自动检测）
--dry-run              # 预览，不写入
--diff                 # 显示差异对比
--no-backup            # 不生成备份
--allow-loss           # GBK 文件：允许不支持的字符替换为 ?（默认拒绝）
--create               # 文件不存在时自动创建
```

## gbktoutf8.py

批量将目录下文本文件从 GBK 转换为 UTF-8，非文本文件（按扩展名判断）直接跳过不复制。

```bash
python gbktoutf8.py <源目录> <目标目录>
python gbktoutf8.py <源目录> <目标目录> --dry-run         # 预览
python gbktoutf8.py <源目录> <目标目录> --with-bom        # 输出 UTF-8 文件加 BOM
python gbktoutf8.py <源目录> <目标目录> --log out.log     # 日志写入文件
python gbktoutf8.py <源目录> <目标目录> --confidence 0.9  # 提高编码置信度阈值
```

**输出状态：** `GBK→UTF8` 成功 / `已UTF-8` 无需转换 / `ASCII` 直接复制 / `跳过` 非文本 / `⚠ 不确定` 需人工核查

---

## utf8togbk.py

批量将目录下文本文件从 UTF-8 转换为 GBK，非文本文件直接跳过不复制。

```bash
python utf8togbk.py <源目录> <目标目录>
python utf8togbk.py <源目录> <目标目录> --dry-run    # 预览
python utf8togbk.py <源目录> <目标目录> --strict     # 遇到不可编码字符时报错（默认替换为 ?）
python utf8togbk.py <源目录> <目标目录> --log out.log
```

**输出状态：** `UTF8→GBK` 成功 / `已是GBK` 无需转换 / `ASCII` 直接复制 / `跳过` 非文本 / `⚠ 有损转` 含不支持字符（已替换为 `?`）/ `⚠ 不确定` 需人工核查

> **注意：** 源目录与目标目录不能相同，目标目录也不能是源目录的子目录。

## read_excel.py

读取 Excel 文件（`.xlsx` / `.xlsm`），支持多种模式和输出格式。

### 六种模式

```bash
# info — 文件概览（工作表列表、行列数、列名）
python read_excel.py <文件>
python read_excel.py <文件> --mode info

# read — 读取数据
python read_excel.py <文件> --mode read
python read_excel.py <文件> --mode read --sheet Sheet2          # 指定工作表
python read_excel.py <文件> --mode read --start 5 --end 20      # 指定数据行范围
python read_excel.py <文件> --mode read --cols 姓名 薪资         # 只读指定列
python read_excel.py <文件> --mode read --limit 10              # 最多输出 10 行

# cell — 读取指定单元格
python read_excel.py <文件> --mode cell --cells A1 B3:C5

# formula — 读取单元格原始公式
python read_excel.py <文件> --mode formula --cells B2:D10

# search — 搜索
python read_excel.py <文件> --mode search --search "张三"
python read_excel.py <文件> --mode search --search "技术部" --col 部门
python read_excel.py <文件> --mode search --search "\d{5}" --regex

# stats — 数值列统计（min / max / 均值 / 求和）
python read_excel.py <文件> --mode stats
python read_excel.py <文件> --mode stats --cols 薪资 利润
```

### 输出格式

```bash
--format table    # 对齐表格（默认）
--format json     # JSON
--format csv      # CSV
--format raw      # Tab 分隔纯数据
--out result.json # 结果保存到文件
```

## write_excel.py

写入 Excel 文件，保留所有未修改单元格的格式与公式。
每次写入前自动备份（`.bak`），原子写入。

### 数据格式（`--data` 支持三种写法）

```bash
--data '[{"姓名":"张三","分数":90}]'   # JSON 对象数组（推荐）
--data '[["张三",90],["李四",85]]'     # JSON 二维数组
--data-file rows.json                  # 从 JSON / CSV 文件读取
```

> 以 `=` 开头的值会被写为 Excel 公式，如 `--cells C1="=SUM(A1:B1)"`

### 十种操作模式

```bash
# set-cell — 写入指定单元格
python write_excel.py <文件> --mode set-cell --cells A1=标题 B2=100 C3="=SUM(A1:B1)"

# append — 追加行
python write_excel.py <文件> --mode append --data '[{"姓名":"张三","分数":90}]'

# insert — 在指定行前插入（--start 为 Excel 行号，含表头）
python write_excel.py <文件> --mode insert --start 3 --data '[["新行",100]]'

# replace — 替换指定行范围
python write_excel.py <文件> --mode replace --start 5 --end 7 --data '[["新内容",200]]'

# delete — 删除指定行
python write_excel.py <文件> --mode delete --start 4
python write_excel.py <文件> --mode delete --start 3 --end 6

# patch — 全局文本替换（支持正则）
python write_excel.py <文件> --mode patch --old "旧文本" --new "新文本"
python write_excel.py <文件> --mode patch --old "v\d+" --new "v2.0" --regex

# set-col — 批量写入整列（{row} 替换为实际行号）
python write_excel.py <文件> --mode set-col --col D --value "固定值"
python write_excel.py <文件> --mode set-col --col E --value "=B{row}*C{row}"

# add-sheet / del-sheet / rename-sheet — 工作表管理
python write_excel.py <文件> --mode add-sheet --sheet-name "汇总"
python write_excel.py <文件> --mode del-sheet --sheet "Sheet3"
python write_excel.py <文件> --mode rename-sheet --sheet "Sheet1" --sheet-name "数据"
```

### 常用选项

```bash
--sheet NAME_OR_INDEX  # 指定操作的工作表（默认第一个）
--dry-run              # 预览，不写入
--no-backup            # 不生成备份
--create               # 文件不存在时自动创建
```

### Excel 行号说明

`--start` / `--end` 使用含表头的 Excel 行号：第 1 行 = 表头，第 2 行 = 第 1 条数据，以此类推。

## 备份说明

`write_gbk.py` 和 `write_excel.py` 每次写入前自动在原文件同目录生成备份：

```
原文件名.YYYYMMDD_HHMMSS.bak
```

可用 `--no-backup` 关闭，`--dry-run` 不生成备份也不写入任何内容。
